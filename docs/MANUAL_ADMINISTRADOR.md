# Manual de administración

Módulos administrativos:

- Usuarios.
- Roles.
- Casos de uso.
- Preguntas requeridas.
- SLA.
- Áreas corporativas.
- Auditoría.

## Usuarios

Permite crear, editar, desactivar, eliminar lógicamente y restablecer contraseñas.

## Roles

Permite definir permisos por módulo: ver, crear, editar y eliminar.

## Casos de uso

Permite administrar el catálogo dinámico por área técnica y severidad.

## Preguntas requeridas

Permite definir preguntas específicas por área, severidad y caso de uso.

## SLA

Permite definir primera respuesta, resolución, horario laboral y pausa de SLA.

## Auditoría

Permite revisar acciones críticas ejecutadas por usuarios.
