# Roadmap recomendado

## Fase 1

- Operación básica de tickets.
- Registro SOS posterior.
- SLA básico.
- Dashboard operativo.

## Fase 2

- Adjuntos físicos con MinIO/S3.
- Envío real por SMTP.
- Reporte PDF/Excel mensual.
- Cierre automático.
- Plantillas de respuesta.

## Fase 3

- SSO/LDAP/AD.
- Integración SIEM/EDR/Firewall.
- Webhooks Teams/Slack.
- MFA para analistas y administradores.
- Motor de reglas de asignación.
- Alembic para migraciones.
